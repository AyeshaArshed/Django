# Django

FirstApp is a Django-based web application that demonstrates URL routing, media handling, and modular app inclusion. This project includes multiple apps like login, books, and student management.
<br>
Login: Handles user authentication and profiles.
<br>
Books: Manages book-related data and CRUD operations.
<br>
Student: Manages student information and records.
<br>
Create a superuser (for admin access, add student data):
<br>
python manage.py createsuperuser
<br>
**Apply migrations:**
<br>
python manage.py makemigrations
<br>
python manage.py migrate
<br>
**Running the Project**
<br>
Start the development server:
<br>
python manage.py runserver
<br>
